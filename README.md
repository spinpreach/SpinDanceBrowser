回転剣舞 (SpinDanceBrowser)
======================

【概　　要】
----------------
DMM.com が配信しているブラウザゲーム「刀剣乱舞-ONLINE-」の専用ブラウザです。

【動作環境】
----------------
+ Windows 7 (32bit, 64bit)
+ Windows 8 (32bit, 64bit)
+ Windows 8.1 (32bit, 64bit)
+ Windows 10 (32bit, 64bit)
+ .NET Framework 4.5 以上
+ Internet Explorer 9

【開発環境】
----------------
+ Visual Studio 2015 (C#)
